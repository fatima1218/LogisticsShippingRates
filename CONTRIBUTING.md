# Contribution Guidelines

Welcome Contributors!  
Thank you for considering contributing to the centralized repository. This document outlines the guidelines for contributing to the development of Shipping Rates and Calculations.

## Code style
Follow the coding conventions used in the existing codebase.

## Documentation
Ensure your contributions are well-documented. Add comments and clear PR descriptions.

## Testing
Test your code thoroughly before submission.

## Issue tracker
Check open issues before starting new work. Comment to claim issues.

## Code review
All submissions will go through a code review process. Be open to feedback.
